# Details

Details shortcode is a helper for `details` html5 element.

In the future, the `expand` shortcode will be replaced by this.

## This example

```tpl
{{</* details title="Title" open=true */>}}

## Markdown content

Here is some awesome content!

{{</* /details */>}}
```

## Renders the following

{{< details title="Title" open=true >}}

## Markdown content

Here is some awesome content!

{{< /details >}}
